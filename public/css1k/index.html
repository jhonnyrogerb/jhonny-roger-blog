Front-end Week

Uma overdose de eventos e palestras sobre front-end em uma semana.

por Diego Eis"27/06/2017

Você deve conhecer o CSS Zen Garden. O CSS Zen Garden foi criado pelo Dave Shea, foi um dos sites responsáveis por convencer toda a galera desenvolvedora orgulhosa e desconfiada mostrando que era possível produzir e implementar websites flexívels, diferentes, não quadrados nem com cara de blog – como falavam – utilizando CSS puro. Tendo este pensamento em mente, Jacob Rask decidiu fazer o CSS1K: um desafio para você que acha que consegue implementar um design utilizando apenas 1K de CSS.

O que você tem que fazer é simples:

– Você só pode enviar CSS.

– O seu código CSS deve ter 1K (1024 bytes), minificado.

– Não precisa utilizar prefixos de browsers, eles colocarão lá para você, logo, você pode usar as coisas bacanas de CSS que só funcionam em browsers (não o IE).

– O Design não precisa ficar igualzinho em todos os browsers, mas gracefull degradation é uma boa pedida.

– Seu código é submetido sob a licensa MIT.

Eles dão o código HTML que você não pode modificar em hipótese alguma, se não acaba a graça da brincadeira.

E aí, consegue?
