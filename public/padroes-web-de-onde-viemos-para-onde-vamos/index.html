Front-end Week

Uma overdose de eventos e palestras sobre front-end em uma semana.

por Diego Eis"27/06/2017

Duas semanas passadas, fui convidado para fazer uma palestra no Palácio dos Bandeirantes, aqui em São Paulo, para um grupo de desenvolvedores web que trabalham para o governo.

Ótima iniciativa e ótimo evento, que vem melhorando e aumentando a bagagem dos profissionais web que trabalham em sites do governo. Pena que nem todos tem este gosto por inovação.

O título da minha apresentação foi: Padrões Web: de onde viemos, para onde vamos?

Onde falei um pouco sobre a história dos Padrões Web. Assunto muito interessante que muitos profissionais desconhecem. Sinceramente, acho uma vergonha um desenvolvedor web não saber o nome do camarada que inventou a web, por exemplo. É a mesma coisa que o camarada trabalhar com carros e não saber quem é Henry Ford.
